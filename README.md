# aushalten.

Eine kleine App für den Umgang mit unangenehmen Gefühlen: jeden Tag eine kurze Übung — halten, atmen, warten oder zeichnen.

## Nutzung

`index.html` direkt im Browser öffnen, kein Build-Schritt nötig.

- `index.html` – Struktur
- `styles.css` – Design
- `app.js` – Aufgaben-Logik, Tagesverlauf, Farbthemen

Fortschritt und gewähltes Thema werden in `localStorage` gespeichert.
